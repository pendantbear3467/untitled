# In-Game Documentation Index

## Machines
- Crusher: Ore to raw resources and baseline tier conversions.
- Pulverizer: Raw ore to dust multipliers for efficient smelting.
- Smelter and advanced processors: Dust and component refinement.

## Abilities
- Core ability icons and definitions live in `data/extremecraft/abilities`.
- Combo mechanic: Speed + Strength increases outgoing damage and grants short absorption.

## Boss Lore
- Ancient Core Guardian: Defense-oriented lab sentinel.
- Void Titan: End-phase void warlord with rage surges.
- Overcharged Machine God: Late-game machine overseer with overload cycles.

## Exploration Hints
- Ancient labs reward research and runic fragments.
- Reactors and towers reward machine parts and ability upgrades.
- Fortress and temple clears are required for top-tier crafting components.
