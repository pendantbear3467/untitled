# Datapack Root

This folder contains both live gameplay datapack content and metadata-only mirror folders. They are not all equal.

Live gameplay owner folders:
- `progression/`
- `recipes/`
- `extremecraft_quests/`
- `classes/`
- `class_abilities/`
- `skills/`
- `skill_trees/`
- `module_abilities/`
- `armor_modules/`
- `tool_modules/`
- `worldgen/`
- `forge/`
- `research/`
- `radiation_sources/`
- `contamination/`
- `contamination_terrain/`

Metadata-only or validation/debug folders:
- `quests/`
- `modules/`
- `materials/`
- `machines/`
- `world_generation/`
- `ec_recipes/`
- `tech_trees/`
- `worldgen_weights/`

Legacy or disconnected folders:
- `skilltrees/`

Common mistake:
- Folder names that look newer or cleaner are not automatically the live owner. Check the Java README in the matching subsystem before adding content.
